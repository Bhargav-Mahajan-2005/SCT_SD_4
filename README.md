# Product Information Web Scraper

## Description
A Python-based web scraper that extracts product information such as product names, prices, and ratings from an e-commerce website and stores the data in a CSV file.

## Features
- Extract Product Names
- Extract Prices
- Extract Ratings
- Save Data to CSV
- Structured Output

## Technologies Used
- Python
- BeautifulSoup
- Requests
- Pandas

## Installation

```bash
pip install requests beautifulsoup4 pandas
```

## Run

```bash
python ratingtocsv.py
```
